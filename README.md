# Proyecto-SPRINT-7

Este proyecto tiene como objetivo desarrollar una aplicación web. En ella se mostrará información del DataFrame vehicles_us.csv, mostrando gráficos sobre anuncios de venta de autos de forma interactiva.

La aplicación web es un dashboard interactivo que puede permitir a los usuarios explorar visualmente datos, generar graficos con la interacción de botones o casillas y acceder desde cualquier navegador una vez que ha sido desplegada.

Link de la aplicación web: https://proyecto-sprint-7-hz0e.onrender.com
